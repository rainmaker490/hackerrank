### Running C Solutions 
```
$ cd Algorithms/<name-of-sub-directory>/
$ gcc -o main <name-of-solution-file>.c
$ ./main < [name-of-custom-input-file] >> define/path/to/output.txt
$ open define/path/to/output.txt
```
